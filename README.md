# ginessential
Gin+Vue，**前后端分离**项目，后端代码  
前端代码：[https://github.com/tobesuperman/ginessential-vue](https://github.com/tobesuperman/ginessential-vue)
## 编译运行
```
$ go build
```
Linux系统或macOS系统：
```
$ ./ginessential
```
Windows系统：
```
$ ginessential.exe
```
## 技术栈
* [gin](https://github.com/gin-gonic/gin)
* [gorm](https://github.com/jinzhu/gorm)
